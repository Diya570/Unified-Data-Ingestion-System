# Unified Data Ingestion System 
A java-based system that ingests TXT,XML,XLS files and stores parsed data into PostgreSQL.

Technologies:
1.Java 17.
2.Java Swing.
3.PostgreSQL.
4.Apache POI.
5.DOM XML Parser.

Features:
1.Multi-format data ingestion.
2.Data preview.
3.File parsing.
4.DataBase storage.
